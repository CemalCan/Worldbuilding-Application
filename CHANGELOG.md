# Changelog

All notable changes to the current Loreforge offline browser prototype will be documented in this file.

The format follows the spirit of [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project uses version sections to group user-facing changes.

## [0.1.0] - Unreleased

### Added

- Offline-first browser app.
- Universe and project dashboard.
- Built-in and custom templates.
- Add from template flow.
- Category field presets and customization.
- Linked entity reference fields.
- Characters, families, locations, events, religions, magic, items, organizations, RPG categories, story planning categories, maps, and more.
- Notes and idea inbox.
- Timeline view.
- Family tree view.
- Relationship overview.
- Relationship graph.
- Story, chapter, and scene planning.
- RPG campaign tools.
- Map board and pins.
- Consistency checker.
- Image upload and image positioning.
- Trash, restore, and permanent delete flow.
- English and Turkish language setting.
- JSON import and export.
- Manual QA checklist.

### Changed

- Improved dashboard.
- Improved README.
- Improved form layouts and card/list layouts.
- Improved modal and destructive action UX.

### Fixed

- Startup and entity creation flow fixes.
- Linked reference import validation fixes.
- Turkish text and symbol rendering fixes.
- Trash scoping fixes.
- Dashboard routing fixes.
- Legacy reference handling fixes.
