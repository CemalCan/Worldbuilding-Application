# Loreforge

Offline-first worldbuilding workspace for writers, game masters, and RPG creators.

## Overview

Loreforge is a dependency-free offline prototype for creating and organizing fictional worlds, stories, and tabletop RPG campaigns.

It runs directly in the browser by opening `index.html`. No backend, account, cloud service, or npm install is required.

Data is stored locally in the browser using `localStorage`. Universes can be backed up or moved manually with JSON import/export.

## Current Features

- Universe/project dashboard
- Built-in templates
- Custom templates
- Add categories from templates
- Custom categories
- Category field presets
- Category field customization
- Linked entity reference fields
- Characters, families, locations, events, religions, magic, items, organizations, RPG categories, story planning categories, maps, and more
- Notes and idea inbox
- Pinned notes, todo notes, spoiler/hidden notes
- Timeline / chronology view
- Family tree view
- Relationship overview
- Relationship graph / connection map
- Story, chapter, and scene planning
- RPG campaign tools
- Map board and pins
- Consistency checker
- Image upload and image positioning
- Trash / restore / permanent delete flow
- English/Turkish language setting
- Themes/settings
- Startup behavior setting
- JSON import/export

## Running Locally

Open `index.html` in a modern browser.

No installation is required.

## Data Storage

Loreforge stores data in browser `localStorage`.

Clearing browser data may delete saved universes. Use JSON export for backups.

Uploaded images are stored locally in the browser. Smaller image files are recommended.

## Suggested Usage

1. Create a universe.
2. Choose a template.
3. Add characters, families, locations, and events.
4. Link entries together.
5. Use the timeline, graph, map board, notes, and consistency checker to keep the project organized.
