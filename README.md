# Hikaye

Hikaye, offline-first çalışan açık kaynaklı bir worldbuilding MVP prototipidir.

## Çalıştırma

Ek bağımlılık gerekmez. `index.html` dosyasını tarayıcıda açmak yeterlidir.

Veriler tarayıcının `localStorage` alanında saklanır. JSON import/export akışı ile evrenler taşınabilir.

## MVP Kapsamı

- Evren oluşturma, düzenleme, soft delete
- Yerleşik ve özel şablonlar
- Kategori, sayfa/entity, etiket, ilişki, backlink ve not sistemi
- Hızlı fikir kutusu
- Yerel arama
- Tema ve görünüm ayarları
- Geri dönüşüm kutusu, geri yükleme ve ikinci onaylı kalıcı silme
- JSON import/export

MVP bilinçli olarak hesap, cloud sync, ödeme, ortak çalışma, backend ve AI entegrasyonu içermez.
