# Security Policy

Loreforge is currently an offline-first prototype that runs in the browser and stores data locally.

## Reporting Security Issues

Please report security issues privately by opening a GitHub security advisory for this repository, or by contacting the maintainer directly if advisory reporting is unavailable. Do not publish exploit details publicly until the issue has been reviewed.

## Safe Use

- Do not open JSON import files from untrusted sources.
- Avoid storing sensitive personal information in the app while it is an early prototype.
- The repository should never include secrets, API keys, tokens, `.env` files, or production credentials.

Thank you for helping keep Loreforge safe.
